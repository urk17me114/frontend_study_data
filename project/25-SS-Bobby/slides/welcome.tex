\section{Organisational Matters}

\begin{frame}{Meetings}
	\begin{itemize}
		\item Weekly meetings
		\item Have slides prepared for every meeting. Present either:
			\begin{itemize}
				\item Current state of work (What did you do? What is still missing?)
				\item Presentation of milestone
			\end{itemize}
		\item One group member should take notes and upload them directly(!) after
		the meeting to \texttt{/minutes}
	\end{itemize}
\end{frame}

  We will split the team in two groups. Each group will receive the same tasks:
  \begin{itemize}
    \item Week 1:
      \begin{itemize}
        \item Do a full requirement analysis for the school system of Bobby's school: What do we need? What do we want? How can we realize this?
      \end{itemize}
    \item Week 2:
      \begin{itemize}
        \item Write a first prototype based on your results from week one.
      \end{itemize}
    \item Weeks 3 \& 4:
      \begin{itemize}
        \item t.b.a.
      \end{itemize}
  \end{itemize}
